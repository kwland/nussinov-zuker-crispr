Topic review

How the two models work

Biology

RNA is single-stranded, so it folds back and pairs with itself. Stems, loops, and bulges decide how the molecule behaves — including whether a CRISPR guide can reach its DNA target.

Computation

Both algorithms are dynamic programming. They solve every short subsequence first, store the answers, and combine them into the answer for the whole molecule. Nothing is recomputed.

CRISPR

For SpCas9 the spacer is 20 nt, and the PAM-proximal seed drives target recognition. If that seed is paired inside the guide's own fold, it may be less available to bind DNA.

Model 1

Nussinov (1978)

Goal: maximize the number of base pairs

N[i,j] = max N[i+1, j] N[i, j-1] N[i+1, j-1] + pair(i,j) maxk N[i,k] + N[k+1,j]
  • Every allowed pair is worth exactly +1
  • O(n³) time, easy to trace by hand
  • Ignores physics — a stack and a lone pair count the same
  • Tends to over-pair, producing unrealistic isolated pairs
Model 2

Zuker (1981)

Goal: minimize free energy (kcal/mol)

V[i,j] = min eH(i,j) hairpin eS(i,j) + V[i+1,j-1] stack eL(i,j,k,l) + V[k,l] bulge / internal WM[i+1,k] + WM[k+1,j-1] + a multiloop
  • Stacking stabilizes; loops cost energy
  • Uses measured nearest-neighbour thermodynamics
  • Isolated pairs are penalized away naturally
  • This is what ViennaRNA / RNAfold implements

Energy model

Watson-Crick nearest-neighbour stacking

Zuker's stabilizing term. Each value is the free energy of stacking one pair directly on another, in kcal/mol at 37 °C. G·C-rich stacks are the most stabilizing — this is why GC content predicts guide behaviour at all.

These ten values are the published Turner/Xia nearest-neighbour parameters. G·U wobble stacks, loop initiation, and the multiloop term in this demo are documented simplifications of the Turner rules — good enough to rank structures, not to quote absolute energies. ViennaRNA remains the reference.

What this project does and does not show

Supported

  • Both recurrences are implemented in full. On GGGAAACCC the Zuker code returns −1.12 kcal/mol, matching the value derived by hand from the stacking table.
  • The tRNA-Phe test: Zuker recovers the accepted cloverleaf — stems of 7, 4, 5 and 5 bp, about 86% positional agreement with the published structure. Nussinov over-pairs the same molecule (29 pairs vs 23) and never forms the four-way junction. Load the tRNA-Phe example in Compare mode to see it.
  • Folding features (self-pairing, seed accessibility, GC) are computed identically for every sequence.

Not yet supported

  • No claim that these features predict guide activity — the known-guide rows are still placeholders.
  • Absolute kcal/mol will not match ViennaRNA, because loop and wobble terms are simplified.
  • Real guide activity also depends on chromatin, repair, off-targets, and Cas9/tracrRNA effects.

Project

About this work

01

Research

Nussinov as a base-pair-count model, Zuker/ViennaRNA as energy-based folding, and CRISPR guide design as the application.

02

Build

Written from scratch, no folding libraries: Nussinov in Python (nussinov.py), and both Nussinov and Zuker in the JavaScript running this page.

03

Apply

Fold guide spacers, extract seed accessibility and self-pairing, and compare known guides against random and bad controls.

04

Communicate

Make the algorithm visible: sequence in, DP matrix and 3D structure out, with the model's limits stated plainly.
